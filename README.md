# Desde el Silencio – Landing Page

A high-converting landing page for a clinical psychologist focused on reducing resistance to therapy and encouraging first contact via WhatsApp.

## 🌱 Features
- Conversion-focused design
- Mobile-first responsive layout
- WhatsApp integration for direct contact
- Trust-building UX (first session explanation, emotional copy)

## 🛠 Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## 🎯 Purpose
This project was built to help women in Latin America take the first step toward therapy by reducing friction, fear, and uncertainty.
